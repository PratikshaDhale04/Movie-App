# CineVerse Pro 🎬

CineVerse Pro is a Netflix-inspired movie streaming web application built with React, PHP, and MySQL/SQLite.

## ✨ Features
- Movie browsing by category
- Search functionality
- User authentication
- Watchlist management
- Reviews & ratings
- Responsive design
- Dark/Light theme
- Admin dashboard

## 🛠 Tech Stack
- React, Bootstrap, Axios
- PHP REST API
- MySQL / SQLite

## 📂 Main Components
- Navbar
- MovieCard
- Banner
- SearchBar
- Review Section
- Watchlist
- Admin Panel

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/cineverse-pro.git
cd frontend
npm install
npm run build
```

## 🗄 Database
- MySQL supported
- SQLite fallback available
